# debugging-vscode

> debug typescript with chrome and vscode

* [x] First compile and generate js file
* [x] Make sure vscode extensions **debugger for Chrome** is added
* [x] start `http-server`
* [x] check `launch.json` debugger config file

```
tsc
http-server
```
